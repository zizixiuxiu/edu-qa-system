## 系统架构

```
用户提问
    ↓
┌─────────────────────────────────────────┐
│  Step1: VL学科识别                        │
│  - Qwen3-VL识别学科                        │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  Step2: 多级RAG检索                      │
│  ├─ T1: 学科库 (8个学科, 18万条)          │
│  ├─ T2: 通用库 (16,443条)                 │
│  └─ T3: 本地知识库 (迭代积累)              │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  Step3: 构建增强Prompt                    │
│  - 融合检索结果                           │
│  - 添加思维链提示                         │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  Step4: LLM生成答案                       │
│  - 本地VL模型推理                         │
└─────────────────────────────────────────┘
    ↓
┌─────────────────────────────────────────┐
│  异步: 云端质检 → 知识提取 → 微调数据      │
│  - 高质量答案存入本地知识库                │
│  - 达到阈值触发LoRA微调                    │
└─────────────────────────────────────────┘
```
