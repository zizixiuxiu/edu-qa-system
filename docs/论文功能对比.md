### 实验验证（100%实现）
1. ✅ **消融实验** - 支持多种实验模式切换
2. ✅ **对比实验** - 基线/RAG/专家路由/完整系统
3. ✅ **可视化展示** - ECharts图表展示实验结果

### LoRA微调工作流
```
Edu-Refiner 系统                    外部开源工具
    │                                    │
    ├─ 知识库积累 ──> 导出JSON/CSV ──> ├─ LLaMA-Factory
    │   (质检后入库)                    │  或
    └─ 质量筛选 (>4.0分)               └─ unsloth
                                           │
                                           └─ LoRA微调训练
                                           └─ 导出Adapter
                                           └─ 加载到系统中
```

**说明：** 系统负责知识积累和质检，通过 `/api/v1/knowledge/export` 导出高质量数据，用户可使用 LLaMA-Factory、unsloth 等开源工具进行 LoRA 微调，然后将微调后的模型加载回系统使用。
