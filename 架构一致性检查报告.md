### 三级检索流程

```python
# Tier 0: 本地高质量知识库 (quality_score >= 4.0)
# Tier 1: 学科知识库 (专家专属知识)
# Tier 2: 通用知识库 (跨学科兜底)

async def retrieve():
    # 1. Tier 0 检索 (权重0.5)
    t0_results = await self._search_tier0(session, query_embedding, subject)
    
    # 2. Tier 1 学科库检索 (权重0.3)
    expert_results = await self._search_expert_knowledge(session, expert_id, ...)
    
    # 3. Tier 2 通用库检索 (权重0.2，条件触发)
    if should_trigger_general:
        general_results = await self._search_expert_knowledge(session, general_expert.id, ...)
```
