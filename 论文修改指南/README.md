# 论文修改指南使用说明

## 📁 文件清单

| 文件名 | 内容 | 用途 |
|--------|------|------|
| `01-删除内容清单.md` | 需要删除的章节和内容 | 第一步使用 |
| `02-修改描述-第3章.md` | 第3章修改详细说明 | 第二步使用 |
| `03-修改描述-第4章.md` | 第4章修改详细说明 | 第二步使用 |

---

## 📝 修改步骤

### 推荐修改顺序

```
第一步 → 第二步 → 第三步 → 第四步 → 第五步
 (删)    (改)     (数据)   (图)     (查)
```

### 详细流程

#### 第一步：删除内容（30分钟）
1. 打开 `01-删除内容清单.md`
2. 按清单删除用户认证、批量导入、人工审核相关内容
3. 调整章节编号

#### 第二步：修改描述（2小时）
1. 打开 `02-修改描述-第3章.md`
2. 按顺序修改3.1.3、3.3.4、3.4.1、3.5.2、3.6.5节
3. 打开 `03-修改描述-第4章.md`
4. 修改第4章相关内容

#### 第三步：更新实验数据（1-2小时）
1. 打开 `04-实验数据更新.md`
2. 启动系统，运行实验
3. 记录实测数据
4. 更新第5章所有数值

#### 第四步：更新图表（1小时）
1. 打开 `05-图表更新.md`
2. 重绘图3-1系统架构
3. 按清单截取6张界面图
4. 更新图表引用编号

#### 第五步：最终检查（30分钟）
1. 打开 `06-快速参考.md`
2. 对照检查清单逐项确认
3. 完成所有勾选

## 📞 修改过程中遇到问题

### 不知道具体在哪一节？
1. 打开论文Word文档
2. 按 `Ctrl+F` 搜索关键词
3. 对照指南中的"位置"说明

### 不确定要不要删除？
- 如果内容包含"用户登录/注册/JWT/权限" → **删除**
- 如果内容包含"批量导入Excel" → **删除**
- 如果内容包含"人工审核界面" → **删除**
- 其他内容 → **保留**

### 实验数据怎么获取？
1. 参考 `04-实验数据更新.md` 中的"实验运行步骤"
2. 或者使用系统当前已有的测试数据
3. 确保数据真实，不要编造

## 💡 修改建议

1. **先删后改**：先删除不需要的内容，再修改保留的内容
2. **分章保存**：每修改完一章就保存一个版本
3. **保留原文**：删除前复制原文到备份文档，便于恢复
4. **一次完成**：建议集中时间一次性完成修改，避免遗漏
