## 数据计算公式

### 贡献度计算
```
专家路由贡献 = expert_routing准确率 - baseline准确率
RAG贡献 = rag_only准确率 - baseline准确率
自我进化贡献 = full_system(3轮) - full_system(初始)
总提升 = full_system(3轮) - baseline
```

### 入库率计算
```
入库率 = 入库知识数量 / 总问答次数 × 100%
```
