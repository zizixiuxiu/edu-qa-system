## 截图操作步骤

### 步骤1：准备系统
```bash
# 1. 清空测试数据（可选）
# 重置数据库或标记为测试数据

# 2. 启动系统
cd backend && python -m uvicorn app.main:app
cd frontend && npm run dev

# 3. 准备测试数据
# - 准备几个学科的问题
# - 上传几张测试图片
# - 运行几个实验
```

### 步骤2：首页截图
```
1. 打开 http://localhost:3000
2. 等待首页完全加载
3. 确保显示：
   - 顶部导航栏
   - 欢迎使用标题
   - 快捷问题按钮
   - 左侧菜单
4. 按 F11 全屏（可选）
5. 截图保存为：fig4-1-homepage.png
```

### 步骤3：问答界面截图
```
1. 点击左侧"智能问答"菜单
2. 在输入框输入测试问题：
   "解方程：2x + 5 = 13"
3. 不要发送，保持输入状态
4. 截图保存为：fig4-2-chat-input.png

5. 点击发送按钮
6. 等待回答完全显示
7. 截图保存为：fig4-3-chat-response.png
```

### 步骤4：图片上传截图
```
1. 在问答界面点击"上传图片"
2. 选择一张测试图片（数学题）
3. 不输入文字，直接点击发送
4. 截图保存为：fig4-4-image-only.png
```

### 步骤5：实验控制截图
```
1. 点击左侧"实验控制"菜单
2. 创建几个测试实验（不同模式）
3. 确保显示实验队列
4. 截图保存为：fig4-5-experiments.png
```

### 步骤6：数据分析截图
```
1. 点击左侧"数据分析"菜单
2. 等待图表加载完成
3. 确保显示：
   - 实验对比柱状图
   - 迭代趋势折线图
   - 专家分布饼图
4. 截图保存为：fig4-6-analytics.png
```
