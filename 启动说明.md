# EduQA 教育问答系统 - 启动说明

## 🚀 快速启动

### 方式1：一键启动（推荐）

双击运行 `start_all.bat`

这会同时启动：
- 后端服务：http://localhost:8000
- 前端服务：http://localhost:5173

### 方式2：分别启动

#### 启动后端

```powershell
# PowerShell
.\start_backend.ps1
```

或

```cmd
# CMD
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

#### 启动前端（新窗口）

```powershell
# PowerShell
.\start_frontend.ps1
```

或

```cmd
# CMD
cd frontend
npm run dev
```

---

## 📋 前置要求

### 后端依赖
- Python 3.8+
- 已安装依赖：`pip install -r backend/requirements.txt`
- 本地 LLM 服务（LM Studio 等）运行在 1234 端口

### 前端依赖
- Node.js 18+
- 已安装依赖：`cd frontend && npm install`

---

## 🌐 访问地址

| 服务 | 地址 | 说明 |
|------|------|------|
| 前端 | http://localhost:5173 | 用户界面 |
| 后端 API | http://localhost:8000 | REST API |
| API 文档 | http://localhost:8000/docs | Swagger UI |
| ReDoc | http://localhost:8000/redoc | API文档 |

## 🧪 快速测试

启动后测试后端是否正常运行：

```powershell
# 测试学科列表
curl http://localhost:8000/experts/subjects

# 测试问答
curl -X POST "http://localhost:8000/chat/send" `
  -H "Content-Type: application/json" `
  -d '{"query": "勾股定理是什么？"}'
```

---

## ❌ 关闭服务

直接关闭对应的命令行窗口即可。

---

## 🔧 故障排查

### 端口占用

```powershell
# 查看占用8000端口的进程
netstat -ano | findstr :8000
```

### 依赖问题

```powershell
# 重新安装后端依赖
cd backend
pip install -r requirements.txt

# 重新安装前端依赖
cd frontend
npm install
```

### 数据库问题

```powershell
# 检查数据库
cd backend
python check_db.py
```
